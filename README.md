# Criando-Um-Organizador-de-Declara-o-de-Imposto-de-Renda
Uma ferramenta através do excel que é capaz de ajudar as pessoas a fazerem a declaração do imposto de renda 

Este aplicativo foi desenvolvido utilizando o Microsoft Excel como plataforma principal, com o objetivo de organizar e centralizar as informações necessárias para a Declaração de Imposto de Renda.

O projeto foi estruturado em múltiplas abas interligadas, incluindo:

Dados do Titular — cadastro das informações pessoais do contribuinte

Informes — registro de rendimentos e fontes pagadoras

Notas — controle de despesas e observações importantes

Tabelas de apoio — padronização de categorias e validações de dados

Foram utilizados recursos nativos do Excel como validação de dados, fórmulas, tabelas estruturadas e organização por campos para facilitar o preenchimento, reduzir erros e tornar a consulta das informações mais rápida.

O resultado é uma ferramenta simples, prática e reutilizável para preparação e conferência de dados antes do envio da declaração.
